---
description: 分析計画を作成する（コーディング前のステップ）
---

データサイエンスの分析計画を作成してください。コードを書き始める前に、以下を明確化または確認します。

1. **目的 (Objective)**: 何を明らかにする分析か？
2. **データソース (Data sources)**: 使用するテーブル / ファイル / API
3. **分析単位 (Unit of analysis)**: 1 行が何を表すか
4. **主要指標 (Key metrics)**: 計算する指標とその定義
5. **リスク (Risks)**: データ品質 / リーク / バイアス / 欠損
6. **検証 (Validation)**: 結果の妥当性確認方法
7. **アウトプット (Outputs)**: 期待される成果物（表 / 図 / レポート / モデル）

計画は **日本語**で記述。プロジェクト固有の文脈は `docs/agent/metrics-and-definitions.md` と `docs/agent/data-catalog.md` を参照すること。

追加の指定: $ARGUMENTS
