# 統計・MLガイドライン

統計分析やML作業で確認すべき項目です。詳細は `.github/skills/statistical-ml-review/SKILL.md` を参照してください。

## 分析開始前に確認する項目

- 目的変数は何か
- 分析単位（1行が何を表すか）
- 対象期間
- サンプル定義と除外条件
- 前提条件
- リーケージリスク

## 統計分析

- 相関と因果を区別する
- 信頼区間や不確実性を報告する
- 過大な主張を避ける
- 実験デザインを文書化する

## 機械学習

- train / validation / test を明確に分離する
- リーケージを防ぐ
- 前処理を文書化する
- シンプルなベースラインと比較する
- 限界を報告する
